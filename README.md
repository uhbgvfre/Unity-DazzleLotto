# Unity-DazzleLotto
Draw number lotto program

### Demo(PC)
- Download `DazzleLotto (PC_Build).zip` and unzip it.
- Execute `DazzleLotto.exe` in DazzleLotto (PC_Build) folder.
- [Project in artstation(vedio)](https://www.artstation.com/artwork/L2KzZ5)
